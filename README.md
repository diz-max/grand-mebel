# Сайт для мебельного центра «Гранд Мебель» в Нягани

Одностраничный сайт для мебельного центра «Гранд Мебель» в городе Нягань (ХМАО).

## Описание

Минималистичный, современный сайт с акцентом на широкий ассортимент мебели для дома, квартиры и офиса. Сайт адаптирован под мобильные устройства и имеет интуитивную навигацию.

## Особенности

- Современный минималистичный дизайн
- Адаптивная верстка (мобильные устройства, планшеты, десктоп)
- Галерея категорий мебели
- Форма обратной связи
- Интерактивная карта (заглушка, готова к подключению Яндекс.Карт или Google Maps)
- Чистый, семантический код

## Структура сайта

1. **Шапка** - логотип и навигация
2. **Герой-секция** - крупный заголовок и призыв к действию
3. **Преимущества** - 4 ключевых преимущества компании
4. **Каталог** - галерея категорий мебели
5. **Контакты** - адрес, телефон, карта
6. **Форма обратной связи** - заказ консультации
7. **Футер** - контактная информация и соцсети

## Технологии

- HTML5
- CSS3 (Flexbox, Grid, CSS-переменные)
- JavaScript (ES6)
- Font Awesome для иконок
- Google Fonts (Montserrat, Open Sans)

## Как использовать

1. Склонируйте репозиторий
2. Откройте файл `index.html` в браузере
3. Для размещения на GitHub Pages:
   - Создайте репозиторий `grand-mebel-nyagan`
   - Загрузите все файлы
   - В настройках репозитория включите GitHub Pages для ветки `main`

## Настройка для продакшена

1. Замените изображения в папке `images/` на реальные фотографии мебели
2. Обновите мета-теги в `index.html` для SEO
3. Подключите Яндекс.Карты или Google Maps в блоке контактов
4. Настройте отправку формы на реальный email или CRM-систему
5. Добавьте аналитику (Яндекс.Метрика, Google Analytics)
